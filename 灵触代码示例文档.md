# 占位符变量

代码区中可直接使用以下占位符，执行时自动替换为实际值：

| 占位符 | 说明 |
|--------|------|
| `{STHWND}` | 原点处的窗口句柄 |
| `{STPID}` | 原点处的进程 PID |
| `{STEXEPATH}` | 原点处进程的 exe 完整路径 |
| `{STPATH}` | 原点处进程的根目录路径 |
| `{STRGB}` | 原点处的 RGB 颜色值 |
| `{STARTXY}` | 手势起始坐标 |
| `{STEXT}` | 当前选中的文本 |
| `{SFILE}` | 当前选中文件路径（单个） |
| `{SFILES}` | 当前选中文件路径列表 |
| `{USERNAME}` | 当前登录用户名 |

> "原点" = 鼠标手势开始时光标所在位置，或手动抓取时点击的位置。
